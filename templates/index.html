Moving average crossover strategy: Trading Signal


A simple app which uses Python 3, Flask , Pandas , and ReactJS using moving averegae crossover strategy. Read it about here and here .

If 'Buy' column is green than it means it's time to buy the security and if 'Sell' column is red it means it is time to sell. The data used here is for three year period. All the rows where they were no 'Buy' or 'Sell' signal was present is left out. The app is built just for fun and should not be used for making real time trading decision.
